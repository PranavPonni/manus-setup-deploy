# MANUS Gloves → Allegro Hand Teleoperation (ROS1, Ubuntu)

## 📌 Overview

This guide sets up **MANUS MetaGloves** on Ubuntu and connects them to an **Allegro Hand** using ROS1.

Pipeline:

```
MANUS Gloves
   ↓
MANUS SDK (Integrated Mode)
   ↓
ROS Bridge Node
   ↓
Retargeting (MANUS → Allegro)
   ↓
Allegro Hand
```

---

## 🖥️ Requirements

* Ubuntu 20.04 / 22.04
* ROS1 Noetic
* MANUS SDK (v3.x)
* Allegro Hand + ROS driver
* USB dongle / bodypack

---

## 🔧 1. Install Dependencies

```bash
sudo apt update
sudo apt install -y \
  build-essential \
  libusb-1.0-0-dev \
  libudev-dev \
  zlib1g-dev \
  libncurses5-dev \
  ros-noetic-desktop-full
```

## 🔌 2. Setup MANUS SDK

### Download & Extract

```bash
cd ~/Downloads
unzip ManusSDK_v3.1.1.zip
```

### udev Rules

```bash
sudo nano /etc/udev/rules.d/70-manus-hid.rules
```

Paste:

```
SUBSYSTEMS=="usb", ATTRS{idVendor}=="3325", MODE:="0666"
SUBSYSTEMS=="usb", ATTRS{idVendor}=="1915", ATTRS{idProduct}=="83fd", MODE:="0666"
KERNEL=="hidraw*", ATTRS{idVendor}=="3325", MODE:="0666"
```

Apply:

```bash
sudo udevadm control --reload-rules
sudo udevadm trigger
sudo reboot
```

## 🔁 6. MANUS → ROS Node

Create file:

```bash
nano src/manus_glove_node.cpp
```

Paste:

```cpp
#include <ros/ros.h>
#include <sensor_msgs/JointState.h>
#include "ManusSDK.h"

int main(int argc, char** argv)
{
    ros::init(argc, argv, "manus_glove_node");
    ros::NodeHandle nh;

    ros::Publisher pub =
        nh.advertise<sensor_msgs::JointState>("/joint_states", 10);

    CoreSdk_InitializeIntegrated();

    while (ros::ok())
    {
        sensor_msgs::JointState js;
        js.header.stamp = ros::Time::now();

        // TODO: Replace with real MANUS data
        js.name = {"index_mcp"};
        js.position = {0.5};

        pub.publish(js);
        ros::spinOnce();
    }

    CoreSdk_ShutDown();
    return 0;
}
```

## ⚙️ 7. Update CMakeLists.txt

Add:

```cmake
include_directories(
  ManusSDK/include
  ${catkin_INCLUDE_DIRS}
)

link_directories(
  ManusSDK/lib
)

add_executable(manus_glove_node src/manus_glove_node.cpp)

target_link_libraries(manus_glove_node
  ManusSDK_Integrated
  ${catkin_LIBRARIES}
)
```

## 🔄 10. Retargeting (Core Idea) - [use manusallegro.py]

Map MANUS joints → Allegro joints:

```cpp
sensor_msgs::JointState cmd;

cmd.name = {
  "joint_0.0","joint_1.0","joint_2.0","joint_3.0",
  "joint_4.0","joint_5.0","joint_6.0","joint_7.0",
  "joint_8.0","joint_9.0","joint_10.0","joint_11.0",
  "joint_12.0","joint_13.0","joint_14.0","joint_15.0"
};

cmd.position.resize(16);

// Example mapping (replace properly)
cmd.position[0] = manus_index_mcp;

pub_allegro.publish(cmd);
```

## ⚠️ Common Issues

❌ Connection refused → run `roscore`
❌ No MANUS data → another Core is running
❌ No Allegro motion → wrong mapping
❌ Permission error → fix udev rules
