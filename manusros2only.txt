# MANUS Gloves → Allegro Hand Teleoperation (ROS2, Ubuntu)

## 📌 Overview

This guide sets up **MANUS MetaGloves** on Ubuntu and connects them to an **Allegro Hand** using ROS2.

Pipeline:

```
MANUS Gloves
   ↓
MANUS SDK (Integrated Mode)
   ↓
ROS2 Bridge Node
   ↓
Retargeting (MANUS → Allegro)
   ↓
Allegro Hand
```

---

## 🖥️ Requirements

* Ubuntu 20.04 / 22.04
* ROS2
* MANUS SDK (v3.x)
* Allegro Hand + ROS2 driver
* USB dongle / bodypack

```bash

```

### udev Rules

```bash
sudo nano /etc/udev/rules.d/70-manus-hid.rules
```

Paste:

```
SUBSYSTEMS=="usb", ATTRS{idVendor}=="3325", MODE:="0666"
SUBSYSTEMS=="usb", ATTRS{idVendor}=="1915", ATTRS{idProduct}=="83fd", MODE:="0666"
KERNEL=="hidraw*", ATTRS{idVendor}=="3325", MODE:="0666"
```

Apply:

```bash
sudo udevadm control --reload-rules
sudo udevadm trigger
sudo reboot
```

## 🔁 6. MANUS → ROS2 Node

Create file:

```bash
nano src/manus_glove_node.cpp
```

Paste:

```cpp
#include <rclcpp/rclcpp.hpp>
#include <sensor_msgs/msg/joint_state.hpp>
#include "ManusSDK.h"

class ManusGloveNode : public rclcpp::Node
{
public:
    ManusGloveNode() : Node("manus_glove_node")
    {
        pub_ = this->create_publisher<sensor_msgs::msg::JointState>("/joint_states", 10);

        CoreSdk_InitializeIntegrated();

        timer_ = this->create_wall_timer(
            std::chrono::milliseconds(20),
            std::bind(&ManusGloveNode::publish_joint_state, this));
    }

    ~ManusGloveNode()
    {
        CoreSdk_ShutDown();
    }

private:
    void publish_joint_state()
    {
        sensor_msgs::msg::JointState js;
        js.header.stamp = this->now();

        // TODO: Replace with real MANUS data
        js.name = {"index_mcp"};
        js.position = {0.5};

        pub_->publish(js);
    }

    rclcpp::Publisher<sensor_msgs::msg::JointState>::SharedPtr pub_;
    rclcpp::TimerBase::SharedPtr timer_;
};

int main(int argc, char** argv)
{
    rclcpp::init(argc, argv);
    rclcpp::spin(std::make_shared<ManusGloveNode>());
    rclcpp::shutdown();
    return 0;
}
```

## ⚙️ 7. Update CMakeLists.txt

Add:

```cmake
find_package(ament_cmake REQUIRED)
find_package(rclcpp REQUIRED)
find_package(sensor_msgs REQUIRED)
find_package(std_msgs REQUIRED)

include_directories(
  ManusSDK/include
)

link_directories(
  ${CMAKE_CURRENT_SOURCE_DIR}/ManusSDK/lib
)

add_executable(manus_glove_node src/manus_glove_node.cpp)
ament_target_dependencies(manus_glove_node rclcpp sensor_msgs std_msgs)

target_link_libraries(manus_glove_node
  ManusSDK_Integrated
)

install(TARGETS
  manus_glove_node
  DESTINATION lib/${PROJECT_NAME}
)

ament_package()
```

## 🔄 10. Retargeting (Core Idea) - [use manusallegro.py]

Map MANUS joints → Allegro joints:

```python
from sensor_msgs.msg import JointState

cmd = JointState()

cmd.name = [
  "joint_0.0","joint_1.0","joint_2.0","joint_3.0",
  "joint_4.0","joint_5.0","joint_6.0","joint_7.0",
  "joint_8.0","joint_9.0","joint_10.0","joint_11.0",
  "joint_12.0","joint_13.0","joint_14.0","joint_15.0"
]

cmd.position = [0.0] * 16

# Example mapping (replace properly)
cmd.position[0] = manus_index_mcp

pub_allegro.publish(cmd)
```

## ⚠️ Common Issues

❌ No MANUS data → another Core is running
❌ No Allegro motion → wrong mapping
❌ Permission error → fix udev rules
❌ Package not found → source `install/setup.bash`
❌ Library not found → export `LD_LIBRARY_PATH`
